# Jumble-O-Launcher | Minimal AF Launcher
AF stands for Ad-Free :D

A fork of [Olauncher](https://github.com/tanujnotes/Olauncher) by [@tanujnotes](https://twitter.com/tanujnotes), with the Jumble feature added by [@PennyS8](https://github.com/PennyS8).

## What's different

Jumble-O-Launcher adds a **Jumble** feature on top of Olauncher's minimal launcher experience.

Long-press any app in the drawer to reveal options. Tap **Jumble** to randomly shuffle that app's sort position in the app list on every refresh. The app's name is unchanged — only its position is randomized. Tap **Unjumble** to restore it to its normal alphabetical position.

You can view and manage your jumbled apps from the Settings page.

This is useful for breaking habitual app-opening patterns by making frequently-used apps harder to find by muscle memory. Every tap is a conscious decision.

## Install

Download the APK from the [latest release](https://github.com/PennyS8/Jumble-O-Launcher/releases/).
